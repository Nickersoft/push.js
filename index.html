Push.js

JavaScript notifications for every browser*

Github NPM Demo

Installing Push is easily installable via NPM:

> npm install push.js

Or, if you prefer, you can use Bower instead:

> bower install push.js

Then, creating a notification is as easy as:

Push.create("Hello world!", {
    body: "How's it hangin'?",
    icon: 'icon.png',
    timeout: 4000,
    onClick: function () {
        window.focus();
        this.close();
    }
});

Go ahead and try it by clicking the "Demo" button above! For more documentation, checkout the official repo.
